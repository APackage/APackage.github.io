ZArchiver

ZArchiver - is a program for archive management. It has a simple and functional interface. Allows you to work with various types of archives: 7zip, zip, bz2, gz2, tar and others.

ZArchiver Features

ZArchiver provides you with the following features:

  • Create archives in the following formats: 7z (7zip), zip, bzip2 (bz2), gzip (gz), XZ, lz4, tar, zst (zstd).
  • Extract archives in the following formats: 7z (7zip), zip, rar, rar5, bzip2, gzip, XZ, iso, tar, arj, cab, lzh, lha, lzma, xar, tgz, tbz, Z, deb, rpm, zipx, mtz, chm, dmg, cpio, cramfs, img (fat, ntfs, ubf), wim, ecm, lzip, zst (zstd), egg, alz.
  • View the contents of archives: 7z (7zip), zip, rar, rar5, bzip2, gzip, XZ, iso, tar, arj, cab, lzh, lha, lzma, xar, tgz, tbz, Z, deb, rpm, zipx, mtz, chm, dmg, cpio, cramfs, img (fat, ntfs, ubf), wim, ecm, lzip, zst (zstd), egg, alz.
  • Create and extract password-protected archives.
  • Edit archives: add/remove files to/from archives (zip, 7zip, tar, apk, mtz).
  • Create and extract multi-part archives: 7z, rar (extract only).
  • Install APK and OBB files from a backup (archive).
  • Partially extract archives.
  • Decompress compressed files.
  • Open archive files from email applications.
  • Extract split archives: 7z, zip, and rar (e.g., 7z.001, zip.001, part1.rar, z01).

Special features:

  • For small files (<10MB) on Android 9 and later: Use direct opening without extracting to a temporary folder whenever possible.
  • Support for multi-threaded reading (useful for multi-core processors).
  • UTF-8/UTF-16 support for file names, allowing the use of national symbols in file names.
