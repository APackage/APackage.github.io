microG Transition Module (Magisk)

This Magisk module helps you remove Google components from your device and replace them with the microG framework.

Installation Steps:

  1. Flash the ZIP file via Magisk (Install from Storage).
  2. Reboot your device.
  3. Manually uninstall the following three Google apps via system settings or using pm uninstall:
    • com.google.android.gms → Google Play Services
    • com.google.android.gsf → Google Services Framework
    • com.android.vending → Google Play Store
  4. Install the microG APK (must succeed).
  5. Install the microG Companion and optionally the Play Store APK.
  6. Install microG Installer Revived.
  7. Reboot your device again.

What does it do?

This Magisk module helps you cleanly remove Google services from your device. You can then install microG, an open-source and privacy-friendly alternative that replicates many Google services like push notifications and maps without tracking you.
